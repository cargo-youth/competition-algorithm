# 这里是线段树的一些解析
## 什么是线段树呢？
+ 这里需要贴一段wiki的解释
+ 线段树是算法竞赛中常用的用来维护 区间信息 的数据结构。
线段树可以在的时间复杂度内实现单点修改、区间修改、区间查询（区间求和，求区间最大值，求区间最小值）等操作。
